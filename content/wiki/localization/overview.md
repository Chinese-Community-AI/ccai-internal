- [Locales & Style](/wiki/localization/locales) — 简体, 繁體, tone, typography
- [Translation Workflow](/wiki/localization/workflow) — keys, review, release

## When to localize

| Surface | Priority |
| ------- | -------- |
| Marketing & events | High — match audience language |
| Product UI | High for CN/TW/HK users |
| Internal wiki | English primary; 中文 summaries optional |
| Legal / safety | Professional translation required |

Partner with **Marketing Bootcamp** for public copy and **Design** for layout (CJK line height, truncation).
