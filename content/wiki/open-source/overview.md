How CCAI maintains public repositories under [Chinese-Community-AI](https://github.com/Chinese-Community-AI).

## Modules

- [Governance](/wiki/open-source/governance) — maintainers, decision-making
- [Releases](/wiki/open-source/releases) — versioning, changelog, security

## Program goals

- Useful tools for the Chinese-speaking AI community
- Clear contribution paths for volunteers
- Sustainable maintainer load (no heroics)

## Repo standards

| Requirement | Detail |
| ----------- | ------ |
| LICENSE | OSI-approved license file |
| README | Install, usage, contributing link |
| CONDUCT | Points to community CoC |
| SECURITY | `SECURITY.md` for vulnerability reports |
