## First contribution (recommended path)

1. Read the [Code of Conduct](/wiki/volunteer-contributor/code-of-conduct)
2. Join community chat (link in [General Onboarding](/wiki/general-onboarding/tools-and-access))
3. Find a **good first issue** on GitHub
4. Comment on the issue before large changes
5. Open a PR with a clear description and test notes

## PR hygiene

- One logical change per PR when possible
- Link issue: `Fixes #123`
- Screenshots for UI changes
- Respond to review within a week (or say you’re blocked)

## Recognition

Contributors are credited in release notes and highlighted in community updates when they opt in.
