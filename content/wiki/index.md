## How to use this wiki

| Step | Action |
| ---- | ------ |
| 1 | New joiner → [General Onboarding](/wiki/general-onboarding/overview) |
| 2 | Find your track → sidebar sections below |
| 3 | Search → **⌘K** or [Search](/wiki/search) |
| 4 | Terms → [Glossary](/wiki/glossary) |
| 5 | Edit content → [Contributing](/wiki/contributing) (developers, via PR) |

Content is **read-only** on this site. Updates ship through git — see [Contributing to the Wiki](/wiki/contributing).
