## Before you benchmark

- [ ] Define task, dataset, and metric (primary + guardrails)
- [ ] Document model version, prompt template, and seed
- [ ] Estimate cost and runtime with Eng / Data

## Reporting checklist

| Item | Include |
| ---- | ------- |
| Baseline | Prior model or human baseline |
| Confidence | Variance across seeds or splits |
| Failure modes | Qualitative examples |
| Repro bundle | Config file or notebook link |

## Anti-patterns

- Cherry-picking a single prompt that flatters your model
- Comparing against unpublished competitor numbers
- Publishing leaderboards without error analysis

Partner with **Data Bootcamp** for dashboards and **Product** for user-impact metrics.
