## Foundations

| Token | Usage |
| ----- | ----- |
| Color | Brand primary, semantic success/warning/error |
| Typography | Headings, body, code (Latin + CJK-friendly stacks) |
| Spacing | 4px grid |
| Radius & shadow | Cards, modals, elevated surfaces |

## Components

Start from shared Figma library components. If you need a new variant, propose it in `#design` before one-off screens multiply.

## Accessibility

- WCAG AA contrast minimum for text
- Focus states on all interactive controls
- Don’t rely on color alone for status
