Core to CCAI’s mission: growing and supporting Chinese-speaking AI builders.

## Modules

- [Events Playbook](/wiki/community-bootcamp/events) — meetups, hackathons, online sessions
- [Moderation](/wiki/community-bootcamp/moderation) — codes of conduct, escalation

## Community channels

| Channel | Norms |
| ------- | ----- |
| Discord / Slack | `#announcements` read-only; support in `#help` |
| WeChat groups | Admin-approved invites; no spam |
| GitHub | Welcoming issues/PRs; label good first issues |

## Success signals

- Event attendance and return rate
- Active contributors (talks, OSS, mentorship)
- Sentiment in surveys and retro notes
