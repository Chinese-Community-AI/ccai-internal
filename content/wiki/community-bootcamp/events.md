## Event types

| Type | Typical size | Lead time |
| ---- | ------------ | --------- |
| Online talk | 50–500 | 2–3 weeks |
| Local meetup | 20–80 | 4–6 weeks |
| Hackathon | 30–200 | 6–10 weeks |

## Run of show (minimum)

1. Venue / stream tech check
2. Welcome + code of conduct reminder
3. Content (talks, hacking, judging)
4. Thanks + feedback survey link
5. Retro within 1 week (internal doc or wiki PR)

## Cross-team coordination

- **Marketing** — promotion timeline and creative
- **Product** — demos aligned with roadmap when relevant
- **Operations** — budget, contracts, insurance if in-person
