Standard tooling for CCAI contributors. Request missing access via Operations.

| Tool | Purpose | Who gets it |
| ---- | ------- | ----------- |
| GitHub (`Chinese-Community-AI`) | Code & docs | All technical roles |
| Slack / Discord | Chat & announcements | Everyone |
| Figma | Design & specs | Design, Product, Marketing |
| Notion / Wiki (this site) | Internal knowledge | Everyone (read); devs (write via PR) |
| Analytics (TBD) | Product & marketing metrics | Product, Marketing, Data |

## Security basics

- Enable 2FA on GitHub and email
- Do not commit secrets; use environment variables
- Report suspicious access to Operations immediately
